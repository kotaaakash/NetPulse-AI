# NetPulse AI

## AI-Powered Network Change Risk & Operations Platform

NetPulse AI helps enterprise networking teams safely plan, review, approve, validate, and monitor infrastructure changes using AI-assisted risk analysis.

---

## Vision

Reduce network outages caused by configuration mistakes by providing intelligent change management.

---

## Features

### Infrastructure Management
- Organizations
- Sites
- Locations
- Racks
- Devices
- Interfaces

### Configuration Management
- Configuration upload
- Automatic versioning
- SHA-256 checksum verification
- Duplicate configuration detection
- Configuration history
- Configuration diff comparison
- Configuration rollback

### Change Management
- Create change requests
- View change requests
- Track change request status
- Associate changes with devices
- Associate changes with configuration versions

### Backend
- FastAPI
- SQLAlchemy ORM
- Alembic migrations
- PostgreSQL
- Pydantic validation
- Swagger/OpenAPI documentation
