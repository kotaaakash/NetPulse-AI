# NetPulse AI

## AI-Powered Network Change Risk & Operations Platform

NetPulse AI helps enterprise networking teams safely plan, review, approve, validate, and monitor infrastructure changes using AI-assisted risk analysis.

---

## Vision

Reduce network outages caused by configuration mistakes by providing intelligent change management.

---

## Features

- Device Inventory
- Network Topology
- Change Requests
- AI Risk Analysis
- Configuration Diff
- Incident Timeline
- Executive Dashboard
